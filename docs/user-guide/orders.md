# Orders

The **Orders** page shows all orders you have placed — quotes that have been submitted and converted to orders.

## Accessing Your Orders

Navigate to your profile menu and select **My Orders**, or click **Orders** in the navigation bar.

![Orders](/img/user-guide/orders/orders.png)

## Order List

Orders are displayed in a table sorted by most recently placed. Each row shows:

| Column      | Description                                      |
| ----------- | ------------------------------------------------ |
| **Order #** | The ERP order number (assigned after submission) |
| **Quote #** | The original SWIFTSELL quote ID                  |
| **Date**    | Date the order was placed                        |
| **Status**  | Current order status                             |
| **Total**   | Order total                                      |

## Viewing an Order

Click any order row to open the order detail view. This is the same quote page but in read-only mode — lines, customer details, and attachments are visible but cannot be edited.

## Returns (RMAs)

If the **RMA feature** is enabled on your platform and you need to return items from an order, a **Create RMA** button will be available on the order page.

See your administrator or contact support for the return process.
