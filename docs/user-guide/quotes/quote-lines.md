# Quote Lines

The **Lines** tab is the main working area of a quote. Each line represents one configured product.

## What Each Line Shows

| Field           | Description                                             |
| --------------- | ------------------------------------------------------- |
| **Product**     | Name and part number of the configured product          |
| **Description** | Configuration description generated by the configurator |
| **Quantity**    | Number of units                                         |
| **Unit Price**  | Price per unit as determined by the configurator        |
| **Line Total**  | Quantity × Unit Price                                   |
| **Image**       | Thumbnail of the configured product (if available)      |

## Adding a Line

Click the **Configure** or **Add Line** button on the Lines tab to open the [Configurator](../configurator) and add a new product to the quote.

## Editing a Line

Click the **edit icon** on any line to re-open it in the Configurator with your previous selections loaded. Make your changes and click **Finish** to update the line.

## Removing a Line

Click the **delete icon** on a line to remove it from the quote. You will be asked to confirm before the line is deleted.

## Quote Total

## Attachments

Below the lines table, the **Attachments** section lets you upload supporting documents to the quote. See [Attachments & Communications](./attachments-and-communications) for details.

:::note
Lines cannot be added or edited once a quote has been converted to an order (type = Order).
:::
