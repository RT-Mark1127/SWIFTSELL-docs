# Quotes Overview

A quote in SWIFTSELL is a collection of configured product line items with pricing, customer details, and supporting information. Quotes are used to present pricing to a customer and ultimately convert to an order.

## Quote Dashboard

The **Quote Dashboard** is your central hub for managing quotes. Access it from the main navigation bar.

### What You'll See

- **Quick Configure** — if your administrator has set up products for fast configuration, they appear here as a product grid. Click any card to immediately open the Configurator and create a new quote.
- **My Quotes** — a table of all quotes you have created, sorted by most recently created. Click any row to open that quote.
- **Quote Requests** — if the Quote Requests feature is enabled, a separate table shows quotes submitted by unauthenticated visitors that are awaiting follow-up.

## Quote Types

| Type              | Description                                                                                |
| ----------------- | ------------------------------------------------------------------------------------------ |
| **Quote**         | A standard pricing document built by a logged-in user                                      |
| **Order**         | A quote that has been submitted and converted to an order                                  |
| **Quote Request** | Created by a public (not logged-in) user via the configurator; requires a rep to follow up |

## Quote Stages

Opening a quote shows a tabbed workflow:

| Stage        | Description                                                               |
| ------------ | ------------------------------------------------------------------------- |
| **Lines**    | The line items on the quote — configured products, quantities, and prices |
| **Customer** | Billing and shipping address, and customer contact information            |
| **Summary**  | A printable overview of the full quote                                    |

Navigate between stages using the stage buttons at the top of the quote, or the left/right arrow buttons.

:::info
The **Summary** stage is disabled until at least one line has been added to the quote.
:::
