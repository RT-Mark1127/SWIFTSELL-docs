# All Quotes

The **All Quotes** view gives System Admins a platform-wide view of every quote created by any user.

## Accessing All Quotes

Go to **Administration → All Quotes**.

## Quote List

Quotes are listed sorted by most recently created. Each row shows:

| Column           | Description                            |
| ---------------- | -------------------------------------- |
| **Quote #**      | The SWIFTSELL quote ID                 |
| **Created By**   | The user who created the quote         |
| **Customer**     | The customer associated with the quote |
| **Status**       | Current quote status                   |
| **Total**        | Quote total                            |
| **Created Date** | Date the quote was created             |

## Opening a Quote

Click any row to open the quote. Admins can view and edit quotes belonging to any user on the platform.

## Filtering and Search

Use the search bar and filters to find quotes by status, date range, user, or customer.
