# App Configuration Types

**App Configuration Types** define the schema for App Configurations — they specify what a configuration value represents and how it should be used by the application.

:::note
Managing configuration types requires **Super Admin** access.
:::

## Accessing App Configuration Types

Go to **Administration → App Configuration Types**.

## Creating a Configuration Type

1. Click **Create Configuration Type**.
2. Enter a name and description.
3. Define the platform(s) this type applies to.
4. Click **Save**.

## Editing a Configuration Type

Click any type row to open and edit it.

## Relationship to App Configurations

Configuration Types act as templates. Once a type is defined, one or more [App Configurations](./configurations) can be created using that type to provide runtime values.
