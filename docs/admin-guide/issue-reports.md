# Issue Reports

The **Issue Reports** section collects error and feedback reports submitted by users from within SWIFTSELL.

:::note
Issue Reports require **Super Admin** access.
:::

## Accessing Issue Reports

Go to **Administration → Issue Reports**.

## Report List

Reports are listed sorted by most recently submitted. Each row shows the reporter, submission date, and a summary of the issue.

## Viewing a Report

Click any row to open the full issue report, which includes:

- The user who submitted it
- A description of the issue
- Any relevant context captured at submission time
