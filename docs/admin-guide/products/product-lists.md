# Product Lists

**Product Lists** control which products are visible to specific groups of users and can be used to apply group-specific pricing.

## Accessing Product Lists

Go to **Administration → Product Lists**.

## Creating a Product List

1. Click **Create Product List**.
2. Enter a name for the list.
3. Assign products to the list.
4. Associate the list with a group or platform.
5. Click **Save**.
