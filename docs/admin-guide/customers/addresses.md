# Addresses

The **Addresses** section contains a master list of all addresses in the system. Addresses are used for billing and shipping on quotes and orders.

## Accessing Addresses

Go to **Administration → Addresses**.

## Address List

Addresses are listed sorted alphabetically. Each record includes street, city, state/province, postal code, and country.

## Creating an Address

1. Click **Create Address**.
2. Fill in the address fields.
3. Click **Save**.

## Editing an Address

Click any address row to open and edit it.
