# Administration Overview

The **Administration** section is the control panel for your SWIFTSELL platform. Access it from the main navigation bar — it is only visible to users with System Admin role level or higher.

## Accessing Administration

Click **Administration** in the main navigation bar. You will see a card grid of all sections available to your role level.
