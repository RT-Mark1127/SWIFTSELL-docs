# Imports

The **Imports** section allows administrators to bulk-import data into SWIFTSELL from files. This is useful for initial data setup or ongoing synchronization of large data sets.

## Accessing Imports

Go to **Administration → Imports**.

## Available Import Types

The Imports dashboard shows four import categories:

| Type                   | Description                             |
| ---------------------- | --------------------------------------- |
| **Products**           | Import product records into the catalog |
| **Customers**          | Import customer records                 |
| **Shipping Addresses** | Import shipping address records         |
| **Users**              | Import user accounts                    |

Click any tile to open that import page.
