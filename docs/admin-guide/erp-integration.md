# ERP Integration

SWIFTSELL integrates with **Infor CloudSuite (CSI)** to synchronize orders, pricing, and product configuration data with your ERP system.

## Accessing ERP Integration Settings

Go to **Administration → ERP Integration**.

## What the Integration Does

When configured, the ERP integration enables:

- Sending submitted orders from SWIFTSELL to the ERP system
- Receiving order confirmation and ERP order numbers back into SWIFTSELL
- Using the Infor CloudSuite configurator engine for product configuration rules and pricing

## Configurator Engine

Products can be configured to use either a **local configurator** or the **Infor CloudSuite (Cloud) configurator**. This is set per-product in the [Products](./products/) administration page via the **Server** field.
