# Navigating SWIFTSELL

## Main Navigation

The top navigation bar is present on every page and provides access to the main areas of SWIFTSELL.

![Navigation bar](/img/getting-started/navigation/navbar.png)

| Area                | Description                                                           |
| ------------------- | --------------------------------------------------------------------- |
| **Home**            | Your starting point after login                                       |
| **Product Catalog** | Browse and search all available products                              |
| **Quotes**          | View and manage your quotes                                           |
| **Orders**          | View your submitted orders                                            |
| **RMAs**            | View, create and manage RMA requests                                  |
| **Profile**         | Manage your account settings                                          |
| **Administration**  | Admin-only section for managing the platform (visible to admins only) |

## Home Page

After logging in you are taken to the **Welcome** screen. This provides quick access to common actions such as starting a new quote or browsing the catalog.

The Welcome screen has three quick-access tiles and one primary action:

| Tile / Button | What it does |
|---|---|
| **Quote Dashboard** | Opens your list of active and recent quotes |
| **Orders** | Opens your submitted orders |
| **Catalog** | Opens the product catalog to start browsing |
| **New Quote** *(center button)* | Creates a new quote and takes you directly into it |

## Search and Filtering

Most list pages (product catalog, quotes, orders) include:

- A **search bar** to filter by name or keyword.
- **Filter controls** to narrow results by category, status, date, and other attributes.
- **Sort options** to change the order of results.
